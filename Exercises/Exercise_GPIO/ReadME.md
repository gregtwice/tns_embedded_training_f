# Exercise on GPIO

In this exercise you will learn step by step how to use GPIO Pins on a STM32 Board.
To achieve that you will use GPIO Pin with a LED and a button.
You will also learn hoy to use interruption.

Turn On the LED

## Question 2

Blink the LED

## Question 3

## Question 4
